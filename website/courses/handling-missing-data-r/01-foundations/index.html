Handling Missing Data in R

Phase 01: Foundations of Missing Data

Missing data is not a nuisance to be swept away with a single line of code — it is a
statistical problem whose correct treatment depends entirely on why the data are
missing. This phase builds the conceptual foundation for the whole course: what missing
data is, how it biases estimates and destroys inferential validity, the formal
MCAR/MAR/MNAR taxonomy from Rubin (1976), how to quantify the cost of missingness via
the fraction of missing information, and the practical R toolkit you will use in every
later phase. By the end you can look at a missing value and reason correctly about which
methods are valid and which will silently mislead you.

Learning Objectives

  • Define missing data and explain how it biases point estimates, reduces power, and can invalidate inference
  • State the formal definitions of MCAR, MAR, and MNAR, and classify a mechanism from a described scenario
  • Explain why the missingness mechanism — not the amount missing — determines which methods are valid
  • Distinguish "fraction of missing data" from "fraction of missing information," and reason about what makes missingness costly
  • Demonstrate, by simulation, that listwise deletion is unbiased under MCAR but biased under MAR
  • Install and orient the core R missing-data toolkit (naniar, VIM, visdat, mice, ggmice, missForest, softImpute)
  • Use base-R missing-data primitives (NA, is.na(), complete.cases()) and distinguish structural from statistical missingness

Prerequisites

Basic statistics (mean, variance, linear regression, expectation, bias). Working R
knowledge (data frames, functions, install.packages()). No prior missing-data theory is
assumed. A recent R (≥ 4.1) with the ability to compile source packages is helpful but not
strictly required.

References

  • Rubin DB. Inference and missing data. Biometrika 1976;63(3):581–592.
  • Little RJA, Rubin DB. Statistical Analysis with Missing Data. 3rd ed. Wiley, 2019.
  • van Buuren S. Flexible Imputation of Missing Data. 2nd ed. CRC Press, 2018. https://stefvanbuuren.name/fimd/
  • Schafer JL, Graham JW. Missing data: our view of the state of the art. Psychol Methods 2002;7(2):147–177.
