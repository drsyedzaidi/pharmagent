Handling Missing Data in R

Phase 02: Diagnosing Missingness in R

Before you impute a single value you must know how much is missing, where it
sits, and why it might be absent. This phase turns the general theory of
missing-data mechanisms (MCAR / MAR / MNAR) into concrete R diagnostics: counting
and summarising missingness, building the shadow matrix so you can test whether
absence in one variable relates to the values of another, visualising co-missingness
patterns, and probing the mechanism with Little's MCAR test — while never losing
sight of the fact that the data alone can never confirm MNAR. By the end you can
produce a reproducible Missing-Data Diagnostic Report for any clinical/PK dataset.

Learning Objectives

  • Quantify per-variable and per-case missingness with base R and naniar verbs
  • Construct and reason about the shadow matrix / nabular data to link missingness in X to observed values of Y
  • Choose the right visualisation (vis_miss, aggr, margin plots, UpSet, geom_miss_point) for each diagnostic question
  • Run and correctly interpret Little's MCAR test, and articulate why MAR and MNAR are indistinguishable from data alone
  • Assemble a full diagnostic workflow into a reproducible report template

Prerequisites

Phase 01 (the taxonomy of missingness: MCAR, MAR, MNAR; the concept of a
missingness indicator). Working R with tidyverse. Install:
install.packages(c("naniar", "visdat", "VIM", "mice", "UpSetR", "ggplot2", "dplyr")).
