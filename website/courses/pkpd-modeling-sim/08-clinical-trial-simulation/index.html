PKPD Modeling and Simulation

Phase 08: Clinical Trial Simulation

Clinical Trial Simulation (CTS) integrates a calibrated PK model, a PD model, and an explicit trial design into a single in-silico experiment that mirrors a planned study, generating virtual patient populations, simulated observations, and summary statistics before a single subject is enrolled. This phase walks from the foundational CTS workflow in mrgsolve through dose-selection by target attainment, rigorous propagation of parameter uncertainty via bootstrap, and ends with a full regulatory-grade PKPD CTS pipeline as a capstone. By the end you can build and defend a simulation-based dose rationale suitable for a regulatory briefing document.

Learning Objectives

  • Assemble a complete CTS workflow: PK model + PD model + trial design + virtual population + simulate + summarize
  • Implement inter-individual variability (IIV) with an omega block and residual unexplained variability (RUV) in mrgsolve
  • Generate dose-response curves by simulation and select doses using target attainment probability
  • Propagate parameter uncertainty via bootstrap resampling and report confidence intervals on PD endpoints
  • Recognize and apply the CTS idioms used in regulatory submissions (FDA/EMA modeling and simulation reports)

Prerequisites

Phases 01–07 of this course, or equivalent: one-compartment oral PK, Emax and indirect-response PD models in mrgsolve, IIV/RUV concepts, and familiarity with dplyr/ggplot2.
