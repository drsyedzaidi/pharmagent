PKPD Modeling and Simulation — From Dose to Effect

A hands-on course on pharmacodynamic (PD) modeling and PK/PD integration using R and mrgsolve. Each lesson derives the math from first principles, builds a working mrgsolve model, and ships a keepable artifact — template, checklist, or analysis script — for use on real projects.

Prerequisites

  • PK foundations: 1- and 2-compartment models, clearance, volume of distribution
  • R basics: dplyr, ggplot2, reading data frames
  • mrgsolve installed: install.packages("mrgsolve")

What You Will Learn

  • How concentration and effect are linked — and why the link is rarely instantaneous
  • Direct effect models: linear, log-linear, Emax, Hill
  • Effect compartment and Ke0 for delayed response
  • Indirect response (turnover) models I–IV
  • Disease progression modeling and placebo effects
  • Tolerance and rebound pharmacology
  • Exposure–response regression for binary and continuous endpoints
  • Clinical trial simulation pipeline in mrgsolve

Tooling

  • R (>= 4.2), mrgsolve (>= 1.0), dplyr, ggplot2
  • Optional: nlmixr2, PKNCA, rmarkdown
  • NONMEM syntax shown in lesson notes where relevant

Pedagogy: Learn → Build → Ship → Check

Every lesson runs the same four-move loop. You derive the concept, implement it in R/mrgsolve, walk away with a reusable artifact, and answer 3 check questions before advancing.
