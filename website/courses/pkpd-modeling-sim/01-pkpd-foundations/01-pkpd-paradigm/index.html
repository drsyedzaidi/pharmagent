Lesson 01: The PKPD Paradigm — From Dose to Effect

Objective. Trace the complete dose → absorption → distribution → effect chain and assign a mathematical role to each link.

Prerequisites. None (first lesson in the course).

Concept (from first principles)

The chain

Every pharmacological intervention traverses four links:

Dose
  │  (Route, formulation, release)
  ▼
Concentration at site of measurement  [PK]
  │  (Link function: Cp → Ce or direct)
  ▼
Concentration at biophase / receptor  [PK-PD link]
  │  (Receptor binding, transduction)
  ▼
Effect                                [PD]

Each link is governed by differential equations. The modeler's job is to identify which equations are needed and parameterize them.

Pharmacokinetic layer

For a one-compartment oral model:

Absorption:
$$\frac{dA_{gut}}{dt} = -K_a \cdot A_{gut}, \quad A_{gut}(0) = F \cdot \text{Dose}$$

Central compartment:
$$\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$$

Plasma concentration:
$$C_p(t) = \frac{A_c(t)}{V}$$

Elimination half-life:
$$t_{1/2} = \frac{\ln 2 \cdot V}{CL}$$

Units discipline (critical): if Dose is in mg, V in L, then $A_c$ is in mg and $C_p$ is in mg/L = µg/mL.
$CL$ is in L/h, $K_a$ in h⁻¹. Always verify units cancel correctly before coding.

Pharmacodynamic layer

The PD model converts $C_p$ (or biophase concentration $C_e$) into an effect $E$.

Three archetypes (explored in depth in Lesson 02):

Type Key feature Example
Direct $E = f(C_p)$ instantaneous Neuromuscular blockade
Effect-compartment (Ce) CCW hysteresis explained by biophase lag (ke0) Sedatives
Indirect response (IDR) Effect on synthesis/elimination of response Corticosteroid on cell counts (warfarin/INR is a related precursor/cascade case)

The simplest direct model is the Emax model:

$$E = E_0 + \frac{E_{max} \cdot C_p}{\text{EC}_{50} + C_p}$$

Parameters:
- $E_0$ — baseline effect (same units as $E$)
- $C_p$0 — maximum drug-attributable change (same units as $C_p$1)
- $C_p$2 — concentration producing 50% of $C_p$3 (same units as $C_p$4)

The Hill (sigmoidal Emax) extension adds cooperativity $C_p$5:

$$E = E_0 + \frac{E_{max} \cdot C_p^\gamma}{\text{EC}_{50}^\gamma + C_p^\gamma}$$

Why the paradigm matters

Before fitting data:
1. Draw the box-and-arrow diagram. Every box is a state variable; every arrow is a rate.
2. Assign units to every parameter. Dimensional errors are the most common mrgsolve bug.
3. Identify the PK-PD link. Direct? Effect compartment? IDR?
4. Choose an identifiable parameterization. You cannot estimate both $C_p$6 and $C_p$7 separately without IV data.

Worked Example

Scenario. An analgesic is dosed orally at 200 mg. PK: CL = 5 L/h, V = 50 L, Ka = 1 h⁻¹, F = 0.8. PD: direct Emax, E₀ = 2 (pain score 0–10), Emax = 6, EC₅₀ = 0.5 mg/L.

Step 1 — Dose entering gut:

$$A_{gut}(0) = 0.8 \times 200 = 160 \text{ mg}$$

Step 2 — Peak concentration:

$$C_{max} \approx \frac{F \cdot \text{Dose}}{V} \cdot \frac{K_a}{K_a - K_e} \cdot \left(e^{-K_e t_{max}} - e^{-K_a t_{max}}\right)$$

With $C_p$8 h⁻¹ and $C_p$9 h⁻¹:

$$t_{max} = \frac{\ln(K_a/K_e)}{K_a - K_e} = \frac{\ln(10)}{0.9} \approx 2.56 \text{ h}$$

$$C_{max} \approx \frac{0.8 \times 200}{50} \cdot \frac{1}{0.9} \cdot \left(e^{-0.1 \times 2.56} - e^{-1 \times 2.56}\right) \approx 2.48 \text{ mg/L}$$

Step 3 — Effect at peak:

$$\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$$0

In this model the score rises with drug effect toward the ceiling $CL$0; at Cmax the effect is 6.99 units. The drug-attributable change at peak is $CL$1, i.e. ~83% of $CL$2 (the maximum drug-attributable change).

(Reference: Gabrielsson & Weiner, §3.1 — direct-link PK/PD model; Rowland & Tozer, Ch. 20 — factors controlling drug effects.)

Build it

The companion script code/lesson_01.R simulates this scenario end-to-end:

  1. Define the mrgsolve model with $CL$3CMT, $CL$4TABLE
  2. Set up a dosing event at t = 0 (200 mg oral, 5-min infusion approximation or bolus into gut compartment)
  3. Simulate over 0–24 h
  4. Plot $CL$5 and $CL$6 on dual-panel ggplot
  5. Annotate Cmax, tmax, and the effect at Cmax

Common Pitfalls

  1. Units mismatch. Dose in mg, V in mL → Cp in mg/mL = 1000 µg/mL. Always track units explicitly; put them in $PARAM comments.

  2. Forgetting bioavailability. In an oral model, A_gut(0) = F * Dose, not Dose. If you initialize gut as Dose you overestimate Cp by 1/F.

  3. Confusing Emax and E_max + E0. The Emax parameter is the incremental maximum. Baseline E0 is separate. A model with E0 = 2, Emax = 6 reaches a ceiling of 8, not 6.

  4. Using linear C–E when saturation is present. If EC50 << Cp, linear approximation fails at high doses. Sigmoid Emax is required whenever the dose range spans the EC50.

  5. **Omitting $CL$7ODEor$CL$8CAPTURE` them, they will not appear in the simulated output dataframe.

Code

lesson_01.R

## ============================================================
## Lesson 01: The PKPD Paradigm — From Dose to Effect
## One-compartment oral PK + direct Emax PD
## ============================================================
## Units: Dose [mg], V [L], CL [L/h], Ka [1/h]
##        Cp [mg/L = ug/mL], Time [h], Effect [pain score 0–10]
## ============================================================

library(mrgsolve)
library(dplyr)
library(ggplot2)

set.seed(42)

# ── 1. Define the mrgsolve model ────────────────────────────────────────────

code <- '
$PARAM
CL   = 5     // Clearance (L/h)
V    = 50    // Volume of distribution (L)
Ka   = 1     // Absorption rate constant (1/h)
F1   = 0.8   // Oral bioavailability (fraction)
E0   = 2     // Baseline pain score
Emax = 6     // Maximum drug-attributable effect reduction
EC50 = 0.5   // Concentration at half-maximum effect (mg/L)
gamma = 1    // Hill coefficient (1 = hyperbolic Emax)

$CMT
// Compartment declarations
GUT    // Oral depot: drug awaiting absorption [mg]
CENT   // Central compartment [mg]

$MAIN
// Apply oral bioavailability to the GUT depot so the absorbed
// amount is F1 * Dose (e.g. 0.8 * 200 = 160 mg). Without this,
// mrgsolve delivers the full nominal dose and ignores F1.
F_GUT = F1;

$ODE
// Rate of change in gut depot
dxdt_GUT  = -Ka * GUT;

// Rate of change in central compartment
// Ke = CL/V; direct elimination from central
dxdt_CENT = Ka * GUT - (CL/V) * CENT;

$TABLE
// Derived quantities computed at each time step
double Cp = CENT / V;             // Plasma concentration (mg/L)

// Sigmoidal Emax model (Hill equation)
// Numerator uses pow() for correct gamma handling
double E  = E0 + Emax * pow(Cp, gamma) /
            (pow(EC50, gamma) + pow(Cp, gamma));

$CAPTURE Cp E
'

mod <- mcode("pkpd_lesson01", code)

# ── 2. Bioavailability scaling ───────────────────────────────────────────────
# F1 is applied to the GUT depot via `F_GUT = F1` in $MAIN (see model above),
# so the absorbed amount is F1 * Dose = 0.8 * 200 = 160 mg.
mod <- param(mod, F1 = 0.8)

# ── 3. Set up dosing event ───────────────────────────────────────────────────
# Single oral dose of 200 mg at time 0
# cmt = 1 (GUT), time = 0, evid = 1 (dose event)
dose <- ev(amt = 200, cmt = 1, time = 0, evid = 1)

# ── 4. Simulate ─────────────────────────────────────────────────────────────
out <- mod %>%
  ev(dose) %>%
  mrgsim(
    start  = 0,
    end    = 24,
    delta  = 0.1,    # 0.1 h resolution for smooth curves
    obsonly = TRUE   # return only observation records
  ) %>%
  as.data.frame()

# ── 5. Find Cmax and tmax ────────────────────────────────────────────────────
cmax_row <- out[which.max(out$Cp), ]
cmax_val  <- cmax_row$Cp
tmax_val  <- cmax_row$time
emax_val  <- cmax_row$E

cat(sprintf("Cmax = %.3f mg/L at tmax = %.2f h\n", cmax_val, tmax_val))
cat(sprintf("Effect at Cmax = %.2f\n", emax_val))
cat(sprintf("t_half = %.2f h\n", log(2) * 50 / 5))

# ── 6. Visualize ─────────────────────────────────────────────────────────────

# Panel A: PK — concentration vs time
p_pk <- ggplot(out, aes(x = time, y = Cp)) +
  geom_line(color = "#2C6FAC", linewidth = 1.2) +
  geom_vline(xintercept = tmax_val, linetype = "dashed", color = "gray50") +
  annotate("point", x = tmax_val, y = cmax_val, size = 3, color = "#E05A2B") +
  annotate("text",
           x = tmax_val + 0.5, y = cmax_val,
           label = sprintf("Cmax = %.2f mg/L\ntmax = %.1f h", cmax_val, tmax_val),
           hjust = 0, vjust = 0.5, size = 3.5) +
  labs(
    title   = "Lesson 01 — One-Compartment Oral PK",
    x       = "Time (h)",
    y       = "Plasma Concentration (mg/L)"
  ) +
  theme_classic(base_size = 13) +
  theme(plot.title = element_text(face = "bold"))

# Panel B: PD — effect vs time (direct link)
p_pd <- ggplot(out, aes(x = time, y = E)) +
  geom_line(color = "#E05A2B", linewidth = 1.2) +
  geom_hline(yintercept = 2,   linetype = "dotted", color = "gray40") +   # E0
  geom_hline(yintercept = 8,   linetype = "dotted", color = "gray40") +   # E0 + Emax
  annotate("text", x = 23, y = 2.3,  label = "E0 = 2",        hjust = 1, size = 3.5) +
  annotate("text", x = 23, y = 8.3,  label = "E0 + Emax = 8", hjust = 1, size = 3.5) +
  annotate("point", x = tmax_val, y = emax_val, size = 3, color = "#2C6FAC") +
  annotate("text",
           x = tmax_val + 0.5, y = emax_val,
           label = sprintf("E(Cmax) = %.2f", emax_val),
           hjust = 0, vjust = 0.5, size = 3.5) +
  scale_y_continuous(limits = c(0, 9)) +
  labs(
    title   = "Lesson 01 — Direct Emax PD (pain score)",
    x       = "Time (h)",
    y       = "Effect (pain score)"
  ) +
  theme_classic(base_size = 13) +
  theme(plot.title = element_text(face = "bold"))

# Combine with patchwork if available, else print separately
if (requireNamespace("patchwork", quietly = TRUE)) {
  library(patchwork)
  print(p_pk / p_pd)
} else {
  print(p_pk)
  print(p_pd)
}

# ── 7. Export figure ─────────────────────────────────────────────────────────
# Uncomment to save:
# ggsave("outputs/lesson01_pkpd_paradigm.png", width = 8, height = 6, dpi = 150)

# ── Output note ──────────────────────────────────────────────────────────────
# Expected output:
#   Cmax ~ 2.48 mg/L at tmax ~ 2.56 h
#   Effect at Cmax ~ 6.99  (drug-attributable change ~4.99 = ~83% of Emax)
#   t_half = 6.93 h
# Two panels: Cp(t) in blue, E(t) in orange, dotted horizontal lines at E0 and ceiling.

Check

For a one-compartment oral model with CL = 10 L/h and V = 100 L, what is the elimination half-life?
  • A. 10 h
  • B. 6.93 h
  • C. 14.4 h
  • D. 3.47 h

Answer: B

t½ = ln(2) × V / CL = 0.693 × 100 / 10 = 6.93 h. Ke = CL/V = 0.1 h⁻¹; t½ = ln2/Ke = 6.93 h.

A drug has E0 = 5, Emax = 5, EC50 = 2 mg/L (Hill γ = 1). At Cp = 2 mg/L, what is the predicted effect E?
  • A. 5
  • B. 7.5
  • C. 10
  • D. 2.5

Answer: B

At Cp = EC50, the drug achieves exactly 50% of Emax. E = E0 + Emax × Cp/(EC50 + Cp) = 5 + 5 × 2/(2+2) = 5 + 2.5 = 7.5.

Which parameter in the sigmoidal Emax model controls the potency (sensitivity) of the concentration–effect relationship?
  • A. Emax
  • B. E0
  • C. EC50
  • D. Hill coefficient γ

Answer: C

EC50 is the concentration that produces 50% of maximum effect and is the standard measure of potency. A lower EC50 means the drug is more potent. Emax controls efficacy (ceiling), γ controls the steepness of the sigmoid, and E0 sets the baseline.

PKPD Paradigm — Reference Card

The Dose-to-Effect Chain

DOSE
 │ F · Dose → GUT compartment
 ▼
PHARMACOKINETICS
 │ Ka: GUT → CENTRAL
 │ CL/V: elimination from CENTRAL
 │ Cp = A_central / V
 ▼
PK–PD LINK
 │ Direct: Cp → E immediately
 │ Effect-cmt: ke0 drives Ce toward Cp (delay)
 │ IDR: Cp modulates Kin or Kout (turnover)
 ▼
PHARMACODYNAMICS
 │ E = f(Ce or Cp)
 ▼
EFFECT (clinical endpoint)

Core Equations with Units

One-Compartment Oral PK

Equation Units check
$\frac{dA_{gut}}{dt} = -K_a \cdot A_{gut}$ mg/h = h⁻¹ · mg ✓
$\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$ mg/h = h⁻¹·mg − (L/h)/L·mg ✓
$C_p = A_c / V$ mg/L = mg / L ✓
$t_{1/2} = \ln(2) \cdot V / CL$ h = L / (L/h) ✓
$K_e = CL / V$ h⁻¹ = (L/h) / L ✓

Emax (Hyperbolic) Model

$$E = E_0 + \frac{E_{max} \cdot C_p}{EC_{50} + C_p}$$

Parameter Meaning Typical units
$E_0$ Baseline (drug-free) effect same as E
$E_{max}$ Maximum incremental drug effect same as E
$EC_{50}$ Potency: Cp at 50% of Emax mg/L (or µg/mL)

Ceiling: $E_{max} + E_0$ as $C_p \to \infty$

Sigmoidal (Hill) Emax

$$E = E_0 + \frac{E_{max} \cdot C_p^\gamma}{EC_{50}^\gamma + C_p^\gamma}$$

$\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$0 Shape
1 Hyperbolic Emax (Clark occupancy)
> 1 Steeper sigmoidal; sharper threshold
< 1 Flatter; broad graded response

Effect Compartment (Biophase)

$$\frac{dC_e}{dt} = k_{e0} \cdot (C_p - C_e)$$

At steady state: $\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$1. Hysteresis loop collapses to a curve when Ce is used as the driver.

Indirect Response (IDR) — Type I (inhibit Kin)

$$\frac{dR}{dt} = K_{in} \cdot \left(1 - \frac{I_{max} \cdot C_p}{IC_{50} + C_p}\right) - K_{out} \cdot R$$

Baseline: $\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$2 (set as initial condition)

Parameter Checklist Before Coding

Question If No → Action
Every parameter has units written down? Annotate $PARAM comments
All rate constants in h⁻¹ (or same time unit as simulation)? Convert or re-parameterize
Dose units match V units? Adjust so Cp = Dose_units / V_units
EC50 units match Cp units? Verify both in mg/L or both in µg/mL
E0 biologically plausible at C=0? Run zero-dose simulation and inspect
IDR baseline set: R0 = Kin/Kout? Set via init() in R, not $INIT

PKPD Model Classification Decision Tree

Does the effect follow Cp with no delay?
  YES → Direct model (Emax or linear)
  NO  → Is there hysteresis on the C–E loop?
          YES → Effect compartment (Ke0 model)
          NO  → Does the drug affect a physiological turnover?
                  YES → Indirect response (IDR) model
                  NO  → Tolerance / transit model (advanced)

Key Numbers for Sanity Checks

  • Oral tmax (1-cmt): $\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$3
  • At EC50: drug achieves exactly 50% of Emax
  • At 10 × EC50: drug achieves 91% of Emax (Hill γ = 1)
  • IDR response peak lag: heuristically of order $\frac{dA_c}{dt} = K_a \cdot A_{gut} - \frac{CL}{V} \cdot A_c$4 after peak Cp (exact lag also depends on the Cp profile)

Reference: Gabrielsson & Weiner §2–4 (PK-PD paradigm); Rowland & Tozer Ch. 20; Holford & Sheiner 1981.
