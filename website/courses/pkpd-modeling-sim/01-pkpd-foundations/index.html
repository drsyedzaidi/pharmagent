PKPD Modeling and Simulation

Phase 01: PKPD Foundations

This phase builds the conceptual and computational scaffolding every pharmacometrician
needs before fitting real data: the dose-to-effect chain, the mathematics of
concentration–effect relationships, and a working R + mrgsolve environment for
simulation. By the end you can translate a verbal PKPD hypothesis into a runnable model.

Learning Objectives

  • Articulate the full dose → PK → PD → effect chain and identify where each model component lives
  • Distinguish direct, indirect, and delayed effect relationships and select the right model structure
  • Write, simulate, and plot a basic PKPD model in R using mrgsolve
  • Apply correct units discipline throughout a PKPD analysis
