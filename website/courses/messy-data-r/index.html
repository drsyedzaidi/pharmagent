Dealing with Messy Data in R — Tidyverse, ggplot2, and Publication Figures

A hands-on course on turning messy real-world datasets into clean, analysis-ready tables
and publication-quality figures using the tidyverse. It starts from tidy data import and
dplyr/tidyr cleaning, moves through relational joins, and builds up the full grammar of
graphics in ggplot2 — colour, scales, accessibility, and multi-panel composition with
patchwork/cowplot. Every lesson is grounded in a pharmacometric/clinical dataset (dosing
records, concentration-time data, covariates, NCA summaries) so the skills transfer
directly to real popPK and PK/PD work, while remaining fully general for any tabular data.

This course was built from the syllabus of the Physalia Dealing with Messy Data in R
workshop (tidyverse import → dplyr/tidyr → joins → ggplot2 → advanced visualization →
patchwork/cowplot multi-panel figures).

Who This Is For

Analysts and researchers with prior R experience who are new to the tidyverse or to
ggplot2-based visualization. No tidyverse background required. Pharmacometricians will
recognize every example dataset; the material generalizes to any tabular science data.

Prerequisites

  • Base R: vectors, data frames, functions, basic control flow
  • No tidyverse or ggplot2 experience required
  • Packages: install.packages(c("tidyverse","patchwork","cowplot","scales","viridis","colorblindr","janitor"))
  • Optional (for pharmacometric examples): mrgsolve, PKNCA

What You Will Learn

  • Import messy CSV/Excel/text data with readr/readxl, and understand tibbles vs base data frames
  • Clean and reshape data with dplyr (select, filter, mutate, summarise, group_by) and tidyr (pivot_longer/pivot_wider, separate, unite)
  • Diagnose and fix common messy-data problems: inconsistent types, whitespace, encoding, duplicate keys, wide-vs-long structure
  • Combine multi-table data correctly with left_join, inner_join, full_join, anti_join, and validate join keys
  • Build any chart from the grammar of graphics: aes(), geoms, statistical transforms, facets, coordinate systems
  • Choose the right plot for the data type (distribution, relationship, composition, time-series/profile)
  • Customize colour, scale, and theme for effective and accessible (colour-blind-safe) storytelling
  • Compose multi-panel publication figures with patchwork and cowplot, and export them at publication resolution

Tooling

  • R (>= 4.2), full tidyverse (readr, dplyr, tidyr, ggplot2, stringr, forcats)
  • patchwork, cowplot for composition; scales, viridis for colour/scales
  • janitor for cleaning helpers; colorblindr/colorspace for accessibility checks
  • Optional pharmacometric context: mrgsolve, PKNCA

Pedagogy: Learn → Build → Ship → Check

Every lesson runs the same four-move loop. You derive the concept, implement it in R,
walk away with a reusable artifact, and answer 3 check questions before advancing.
