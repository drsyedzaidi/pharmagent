Dealing with Messy Data in R

Phase 04: ggplot2 Grammar of Graphics

This phase builds every chart in the course from one consistent mental model — Wilkinson's
grammar of graphics as implemented in ggplot2 — instead of memorizing one-off recipes per
chart type. You will construct a plot layer by layer starting from a bare ggplot(data, aes(x,y)) call, learn to choose the right geometry for a given data type and question, and
control facets, statistical transforms, and coordinate systems — including the semi-log
concentration-time plot that is the default display format for PK data. By the end you can
look at any tabular dataset and know which geom, which scale, and which panel layout the
data actually calls for.

Learning Objectives

  • Decompose any ggplot2 chart into its grammar components: data, aesthetic mappings, geoms,
    stats, scales, coordinate system, facets, and theme
  • Distinguish mapping an aesthetic to a variable (aes(color = ID)) from setting it to a
    constant (color = "steelblue"), and predict the visual result of each
  • Build a fully layered PK concentration-time plot: individual points, connecting lines, and
    a summary ribbon, all in one ggplot2 call chain
  • Apply a decision framework — distribution, relationship, composition, categorical
    comparison, time-series/profile — to choose the correct geom for a given variable pair
  • Build the PK spaghetti-plot idiom: per-subject lines at low alpha plus a bold central-
    tendency overlay via stat_summary()
  • Use facet_wrap() and facet_grid() to build small multiples across a categorical
    stratum (e.g. dose group)
  • Distinguish geom-internal stats (geom_boxplot, geom_histogram) from explicit stat
    layers (stat_smooth, stat_summary)
  • Choose the correct coordinate system and scale transform for the data — including
    scale_y_log10() for semi-log PK plots — and explain why coord_cartesian(ylim = ...)
    is safe for zooming while ylim()/scale_*_continuous(limits = ...) silently drops data
    and biases any fitted smooth

Prerequisites

Phases 01–03 (tidyverse import, dplyr/tidyr cleaning, joins). Base familiarity with data
frames/tibbles. install.packages(c("tidyverse", "scales")) (ggplot2 ships inside
tidyverse).

Continuity Dataset

All three lessons simulate and reuse the same population PK dataset: 24 subjects split
across three oral dose groups (50/100/200 mg), one-compartment first-order absorption and
elimination with between-subject variability on Ka, Ke, and V, plus proportional and
additive residual error, sampled at 9 nominal timepoints (0–24 h). Each lesson's script
regenerates it from the same set.seed() so every plot is reproducible standalone.
